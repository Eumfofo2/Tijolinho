# Cassino - Jogo do Tijolinho

## Overview
A Brazilian Portuguese casino-style slot game ("Jogo do Tijolinho") built with vanilla HTML, CSS, and JavaScript. Players place bets and spin three slots to match symbols for bonus multipliers.

## Project Architecture
- **Frontend**: Static HTML/CSS/JS (no build step required)
- **Entry point**: `index.html`
- **Game logic**: `main.js`
- **Styles**: `style.css`
- **Assets**: `imagens/` directory (background and decorative images)

## How to Run
- Served via `npx serve` on port 5000
- No backend or database required
